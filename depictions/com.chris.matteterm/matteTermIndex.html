Compatible with iOS 8 ~ 11

Initial code base and project from @CodeHex via White Terminal. Reworked and updated by me @ChrisOSX into MatteTerm for iOS 8 - 11.


Main Features

Custom Colors:
  • Choose between light or dark mode, or mix 'em up!
  • Many themes included or you can simply build to your liking.

    • 4 Terminal Windows:
  • Do up to 4 tasks at once, even as different users


    • Additional Features

    Configurable Shortcuts Menu:
  • Automate common commands.
  • Create and save your own commands.

    • Optional Mail & Default-Only Keyboards:
  • Also supports 3rd party keyboards & KB dismissal for a larger view.

    • Large Selection of Great Fonts:
  • You are no longer stuck with a font from the 1970's!

    • Useful Toolbar:
  • Features arrows, CTRL key, commonly used CLI keys & more.

    • Paste Functionality:
  • Copy commands from anywhere & paste them right into MatteTerm, (Still in development for better response).

    • Intuitive Gestures:
  • Several options to toggle using simple gestures.


  • iOS 11.3.1 compatibility
  • Misc code clean up.
