Compatible with iOS 8 ~ 10

An amazing forum app specifically designed app for our forum and its members at insanelyi. Started off as a small project, but through tinkering here and there, it has become something pretty amazing. I hope you enjoy it!


Main Features

Navigation Views:
  • Left to right reveals your menu, along with Settings, Notes and My Files buttons at the bottom.
  • On the main screen, tapping the chat bubble icon reveals the Babble view. A dedicated view of the Babble box, similar to the home page. Navigate to your hearts content and still be able to tap the icon and stay in the convo!

    • Settings View:
  • Contains power options such as: Respring, Reboot and Safemode.
  • With additional buttons for clearing Cookies and UICache.

    • Switches in Settings:
  • Pull to Refresh.
  • Toast Messages.
  • Dark Keyboard.
  • App Sounds.
  • Allow Landscape (Videos will still rotate if turned off).
  • Swipe Keyboard. (Enables the ability swipe the keyboard away).
  • 2 finger nav. (Changes the screen edge swipe gesture to 2 fingers, helps with certain 3rd party tweaks).

    • My Files View:
  • Two part view with Active Downloads and Completed tabs.
  • Active view allows you to track the progress of your files. Supports multiple downloads!
  • Completed view allows you to share, unzip files (unzip must be installed from Cydia), and install debs.


    • Additional Features

  • App will never exit when closed. Not like the old one where it gets killed in apples 10 min limit.
  • Implemented a Notes style view located in the Left menu on the bottom.
  • Implemented WKWebView which uses the javascript engine like in Safari.
  • Redesigned image viewer. Flick or tap image away like Tweetbot. Long press for actions. Image viewer now works with Gallery. Downloading wallpapers and saving to photos app.
  • Upgraded download manager, works better with some files. Send to other apps, like iFile, Dropbox etc...
  • Pull to refresh along with pull and release sounds.
  • Scroll to top.
  • Redesigned 'Forum like icons' in the left menu.
  • No connection detection Changed it to dismiss instead of Close App so you can still access downloaded files.
  • New app Alert view animations and style.

    • When you install the app for the first time, sign in from the main screen. Then tap the chat bubble icon. Then it's just a simple pull to refresh in the Babble view should sign you in.
